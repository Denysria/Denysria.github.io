RIA.com делится с вами самым ценным: новый портал API

  • Дата публикации: 02.08.2017

С API каждый пользователь имеет возможность взять для своего сайта или приложения часть информации и данных RIA.com Вам не нужно разрабатывать все с нуля — мы уже сделали это и готовы делиться с вами.

Какую информацию вы можете получать с помощью API RIA.com уже сейчас?

  1. Средняя цена на автомобили
  2. Поиск продаваемых авто
  3. Список типов транспорта, кузова, марки и модели авто, города и другое
  4. Информация о конкретном объявление по ID
  5. Cписок всех областей, городов и районов Украины
  6. Изменение статуса заказа
  7. Получение списка заказов

Воспользуйтесь нашими API и получите доступ к уникальной статистике, данным и запросам.

Мы разрабатываем также персонализированные API. С их помощью вы можете автоматизировать ежедневную работу: меняйте текст и параметры объявлений, удаляйте и добавляйте новые, просматривайте статистику и много другое. Вы сможете управлять объявлениями не только на AUTO.RIA, но и на DOM.RIA и RIA.com.

Сообщите нам, что вам будет полезно для работы и мы учтем все пожелания при разработке новых API. Сделаем вместе портал с API еще лучше.

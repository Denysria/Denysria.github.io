Документация

Используйте эту документацию для доступа к ресурсам и их использования через интерфейсы прикладного программирования (API) . Для использования этих API вам понадобится ключ. Зарегистрироваться для получения API key.

Единственным обязательным требованием для использования сервисов является наличие ссылки на используемый сервис проекта (AUTO.RIA, DOM.RIA, RIA.com) с гиперссылкой на страницу проекта, (AUTO.RIA, DOM.RIA, RIA.com) не закрытой для индексации поисковыми системами

Общая Информация

  • Использование API Key - Как использовать свой ключ API после регистрации.
  • Лимиты Веб-Сервиса - Ежедневные и часовые ограничения на доступ к веб-сервисам RIA.com.
  • Ошибки Веб-Сервиса - Распространенные коды ошибок, которые могут быть возвращены веб-сервисами RIA.com.
  • API Поиска транспортных средств - С помощью данного веб-сервиса можно осуществлять поиск транспортных средств, а результат получить в формате json
  • AUTO info Информация По id Объявления - Дает возможность получить информацию по id Объявления в формате json
  • Подсчёт средней цены - Сервис с помощью которого можно узнавать актуальные средние цены,следить за изменениями цен,анализировать и прогнозировать изменения цен.
  • AUTO.RIA Search Bot - Экспериментальный Telegram-бот написанный на nodejs, для демострации возможностей API поиска AUTO.RIA.
